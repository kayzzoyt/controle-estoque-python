# controle-estoque-python

Sistema simples de controle de estoque desenvolvido em Python.

O programa roda no terminal e permite cadastrar produtos, atualizar informações, excluir itens e visualizar o estoque atual através de um menu interativo.

Os dados são armazenados em memória utilizando estruturas de dados da própria linguagem.

Projeto desenvolvido como atividade acadêmica para prática de lógica de programação, funções e manipulação de dicionários em Python.
